# FLAT CUT - FINAL PRODUCT SPECIFICATION

## PRODUCT DEFINITION
Flat Cut is a single-material flat-cut sign to custom dimensions. Dimensions are primarily width and height, with the material's intrinsic thickness defining the sign's thickness. The material is NOT bonded to any additional substrate we sell.

---

## 1. MATERIALS & THICKNESS

| Material | Thickness | Notes |
|----------|-----------|-------|
| **ACM** | 3mm | Standard for flat signage |
| **Back Black ACM** | 3mm | Gloss black reverse (ideal for fence/gate installations where reverse is visible) |
| **Acrylic** | 3–20mm | Laser cut (in-house or subcontracted for complex shapes) |
| **PVC** | 10–30mm | CNC cut; minimum letter size 100mm (bold typeface) |
| **Aluminium / Metals** | 1.6–3mm | Custom consultable; uncommon |

---

## 2. FINISHING OPTIONS

**Three core finishing approaches:**

### Option A: RAW / DIRECT
- Material with no finishing
- Cut (CNC or Laser) → Ready
- Workflow: Cut → Done

### Option B: UV PRINTED + LAMINATION
- Full colour printing via flat bed UV printer (max 1340mm width × custom length per roll)
- Lamination applied immediately after printing (Matte OR Gloss - not both)
- Clear coat enamel (optional, applied after cutting - super fine coating, all faces)
- Workflow: Print → Laminate → Cut → (Optional) Clear Coat Enamel

### Option C: VINYL WRAPPED + LAMINATION
- Vinyl pre-printed with Epson ecosolvente (max 1340mm width × custom length per roll)
- Lamination applied after printing (Matte OR Gloss - not both)
- Material base cut (ACM, Acrylic, or PVC)
- Vinyl wrapped manually using lamination table + roller (wraps around edge)
- Workflow: Print vinyl → Laminate vinyl → Cut material → Apply vinyl

**Key Decision:**
- **UV Printed path**: Choose EITHER Lamination OR Clear Coat Enamel (not both)
- **Vinyl Wrapped path**: Always includes Lamination (Matte or Gloss)

### Color Specification
- Full colour always (for all printing options)

### Lamination Specifications
- Minimum thickness: 3mm
- Finish options: Matte OR Gloss (mutually exclusive per piece)

---

## 3. CUT SHAPES & FORMATS

**Available formats:**
- Rectangular panels
- Custom shapes
- Letters (flat cut - not 3D)

### CNC Specifications (ACM, Back Black ACM, PVC)
- Maximum: 1200×2400mm
- Bit diameter: 6mm
- Minimum interior radius: 3mm
- Final minimum detail size: ~100mm (when thickness > 10mm)

### Laser Specifications (Acrylic)
- Maximum: 1200×2400mm
- Tolerances: ±0.1–0.3mm
- Complex shapes may be subcontracted; we handle finishing

### Letter Size Recommendations

| Material | Thickness | Minimum Size | Visual Appearance |
|----------|-----------|--------------|-------------------|
| **ACM / Back Black** | 3mm | ~50–80mm | Flat |
| **Acrylic** | 3–10mm | ~50–100mm | Flat (guaranteed) |
| **Acrylic** | 10–20mm | ~75–150mm | Transition to 3D-like (varies by total letter size) |
| **PVC** | 10–30mm | 100mm (bold) | >10mm reads as 3D-like |

**Fonts:** Custom designs only (no pre-loaded standard fonts)

### Visual Appearance by Thickness

| Thickness | Appearance |
|-----------|-----------|
| < 10mm | Clearly flat and thin |
| 10–15mm | Transition zone (depends on total letter/shape size) |
| > 15mm | 3D-like appearance |

*Special case: A 1m tall letter in PVC 30mm may still read as "flat" due to scale*

## 4. MOUNTING OPTIONS

All available:
- Adhesive backing
- Pre-drilled holes with threads
- Standoffs
- Screws (recommended for exterior + high-traffic areas for better support)

---

## 5. LOCATION & DURABILITY CONSIDERATIONS

### Material Recommendations by Environment

**Interior:**
- All materials fully compatible
- All finishing options available

**Exterior:**
- **PVC:** NOT recommended (fades/discolours under UV exposure)
- **UV Printed finish:** Deteriorates with high sun exposure → Vinyl wrapping superior UV durability
- **Vinyl wrapped:** Recommended for exterior (superior UV resistance)
- **Back Black ACM:** Especially suitable for fences/gates (reverse face visible; gloss black finish)

### Location Parameter (Required for order)
- Interior
- Exterior
  - If exterior, note orientation (North-facing = maximum solar incidence in Australia)

---

## 6. PRODUCTION CAPABILITIES & SUBCONTRACTING

### In-House Production
- Flat bed UV printing: 1340mm width × custom length per roll
- Epson ecosolvente vinyl printing: 1340mm width × custom length per roll
- Vinyl lamination: Full roll capacity, Matte or Gloss finish
- CNC cutting: 1200×2400mm, 6mm bit
- Laser cutting: Acrylic (in-house or subcontracted for complex shapes)

### Subcontracting
- **Acrylic cutting** (complex shapes): We subcontract the cut only
  - Client specifies shape → We print & laminate → Receive cut form → Apply vinyl (if applicable)
  - After subcontracted cut, we can finish the material in-house
- **Custom sizes beyond capacity**: Quoted separately; always include printing

### Large Panels Without Printing
- Possible (custom sizes outside standard capacity)
- Require separate quotation
- Pricing varies

---

## 7. VINYL WRAPPING DETAILS

### Application Process
- Vinyl is laminated after printing (Matte or Gloss)
- Material base is cut to shape
- Vinyl is wrapped manually using lamination table + roller
- **Vinyl wraps around the edge** of the cut material (no gaps)
- We cut and apply the vinyl (exact fit to cut shape)

### Compatibility
- Compatible with all material thicknesses
- Partial vinyl wrapping = classified as **Decal** product (separate offering)

## PRODUCTION WORKFLOW SUMMARY

### **Workflow 1: Raw**
```
Choose material & cut shape → CNC/Laser cut → Mounting hardware → Done
```

### **Workflow 2: UV Printed + Lamination**
```
Design → Print UV (1340×custom) → Laminate (Matte/Gloss) → Cut (CNC/Laser) → Mounting hardware → Done
```

### **Workflow 3: UV Printed + Clear Coat**
```
Design → Print UV (1340×custom) → Cut (CNC/Laser) → Clear coat enamel (manual, all faces) → Mounting hardware → Done
```

### **Workflow 4: Vinyl Wrapped + Lamination**
```
Design → Print vinyl (Epson, 1340×custom) → Laminate (Matte/Gloss) → Cut material base → Apply vinyl (manual + lamination table) → Mounting hardware → Done
```

### **Workflow 5: Acrylic (Complex Shape - Subcontracted)**
```
Design → Print (UV or Epson) → Laminate → Subcontract laser cut → Receive cut acrylic → Apply vinyl (if applicable) → Mounting hardware → Done
```
