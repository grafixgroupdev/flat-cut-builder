## 🚀 KEY SELLING POINTS

**For Exterior Durability**: Vinyl Wrap + Gloss Laminate
- Superior UV resistance
- 5-7 year lifespan (vs 2-3 for UV print alone)

**For Budget**: ACM 3mm + UV Print + Laminate
- Cost-effective
- Full color
- Interior/exterior capable

**For Premium**: Acrylic + Vinyl Wrap
- High-end appearance
- Best for complex shapes
- Subcontracting available

**For Speed**: ACM Raw or UV Print only
- 3-5 day turnaround
- No lamination delay

---

## ⚠️ CRITICAL CONSTRAINTS

| Constraint | Impact |
|-----------|--------|
| **PVC = Interior Only** | Exterior = NO (UV decolouration) |
| **Vinyl = Exterior Best** | UV Print deteriorates outdoors |
| **1340mm max width** | Larger sizes need custom quote |
| **Laminate OR Clear Coat** | Not both (choose one path) |
| **Acrylic complex shapes** | May require subcontracting (5-10 day lead) |

```
Design → Print Vinyl (Epson) → Laminate (M/G) → Cut Material → Apply Vinyl → Mount → Ship
```

## 🎓 MATERIAL SELECTION QUICK GUIDE

**Start with LOCATION → then Material**

### Interior
- **Budget**: ACM (any finish)
- **Premium**: Acrylic + UV Print
- **Specialty**: PVC for thick letters (10-20mm)

### Exterior
- **Standard**: ACM + UV Print (OK but not ideal)
- **Best**: Acrylic + Vinyl Wrap
- **Fancy**: Back Black ACM for fences (gloss black reverse)
- **Avoid**: PVC (will fade)

### Exterior + High UV (North-facing Australia)
- **Strongly Recommend**: Vinyl Wrap (any material)
- **Second Choice**: Acrylic + UV Print + Clear Coat

## 📞 WHEN TO ESCALATE

**To Production Team:**
- Subcontracting required (acrylic complex shapes)
- Custom sizing >1340mm width
- Rush delivery <3 days
- Material not listed (custom metals)

**To Design Team:**
- Letters <50mm (too small, suggest increase)
- Complex logos (suggest simplification)
- Tight tolerances (clarify requirements)

**To Client (before quote):**
- "High UV area + UV Print only?" → Recommend Vinyl upgrade
- "PVC + Exterior?" → Educate on decolouration risk
- ">1340mm wide?" → Confirm multi-panel approach
