# FLAT CUT - VENDOR QUICK CARD

## CONFIGURATION STEPS (In Order)

1. **SHAPE** → Panel / Custom Shape / Letters
2. **DIMENSIONS** → W × H (mm) - System auto-flags if subcontracting needed
3. **MATERIAL** → ACM / Back Black / Acrylic (3-20mm) / PVC (10-30mm)*
4. **LOCATION** → Interior / Exterior
5. **FINISHING** → Raw / UV Print+Laminate / UV Print+Clear Coat / Vinyl Wrap+Laminate
6. **FINISH TYPE** → Matte OR Gloss (if laminate)
7. **MOUNTING** → Adhesive / Pre-drilled / Standoffs / Screws
8. **NOTES** → Font, rush, orientation, special requests

*PVC = Interior only (UV decolouration risk)
**Complex cutting automatically determined by dimensions (>1200×2400 or complex geometry)

## FINISHING PATHS

**RAW**
- Cut only
- Cheapest
- Interior only
- ~3 days

**UV PRINT + LAMINATE**
- Full color printing (1340×custom max)
- Matte or Gloss
- Good durability indoors
- Clear coat OPTIONAL
- ~3-5 days

**UV PRINT + CLEAR COAT**
- Full color printing
- Super-fine protective coat (all faces)
- No laminate
- Interior preferred
- ~3-5 days

**VINYL WRAP + LAMINATE**
- Print vinyl (1340×custom max)
- Matte or Gloss
- **BEST for exterior**
- Max UV durability
- ~5-7 days (may subcontract)

---

## RED FLAGS TO WATCH

⚠️ **PVC + Exterior** → Redirect to Acrylic + Vinyl
⚠️ **UV Print + North-facing + Outdoor** → Suggest Vinyl upgrade
⚠️ **>1340mm width** → Confirm custom/paneling required
⚠️ **Complex shape + tight timeline** → Warn about subcontracting delay
⚠️ **Outdoor + UV Print only** → Will deteriorate; recommend Vinyl
⚠️ **Letters <50mm** → Too small; suggest design adjustment
